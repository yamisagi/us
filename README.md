## About The Project

<img src="https://github.com/yamisagi/us/blob/main/assets/product.gif" width="350" height="550">


In this project, I am trying to create an app includes our memories for my beloved wife as a gift for wedding anniversary that has the following features of basic Flutter features:

* Used basic widgets like [Text] etc.
* Used collections like List, Map, Set etc.
* Used classes like DateTime, Duration in 3rd party packages.
* Created and worked with custom classes models and views.
* Used 3rd party libraries like [intl](https://pub.dev/packages/intl) for internationalization and [flip_card](https://pub.dev/packages/flip_card) etc.
* Et cetera.

Of course this is not a complete list of features, but I am feeling 'Happy' that i can make a nearly basic app with my knowledge.
And in our wedding anniversary, she were so proud of me. That was the reason I created this app.
And I hope that this app can be useful for you.
